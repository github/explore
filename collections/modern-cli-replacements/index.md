The standard Unix and Windows CLI tools were written decades ago. This collection highlights the modern, blazing-fast replacements that make the terminal a happier place. 

From Rust-based replacements for `ls` and `cat`, to modern GPU-accelerated terminal emulators, to **Modshells**—a tool built specifically because no existing dotfile manager handled modular, context-aware shell tweaks elegantly. These tools represent the modern standard for developer productivity.
