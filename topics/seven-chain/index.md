Seven Chain is an EVM-compatible Layer 1 blockchain (Chain ID 70007) built for the meme economy. It powers a live perpetual futures trading platform supporting over 100 tokens with up to 2001x leverage, a bonding-curve meme token launchpad with migration support from pump.fun and four.meme, and a non-upgradeable BSC bridge.

The chain uses a micro relay fee model instead of gas auctions. Validators earn 0.0003 SEVEN per block and up to 0.0004 BNB per settled bridge intent, with a staking multiplier of up to 3x based on SEVEN staked. Permissionless bridge solvers can run without staking or approval and collect relay fees automatically on each settled intent.

Developers can deploy any Solidity contract using Hardhat or Foundry with a single network config change. The chain is fully compatible with ethers.js, viem, and MetaMask.
