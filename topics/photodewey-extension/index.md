PhotoDewey is a free photo library manager for Windows. It lets photographers organize, tag, and edit their photos without ever modifying the original files. The built-in editing tools cover common adjustments like tone, exposure, filters, and rotation — and the extension system lets developers add more.

A `photodewey-extension` is a community-built plugin that integrates directly into PhotoDewey. Extensions can add new editing operations that appear alongside the built-in tools, or add informational panels to the sidebar. Users install an extension by dropping it into a folder; PhotoDewey picks it up automatically the next time it starts.
