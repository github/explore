Master Skill (大师.skill) turns any Claude Code agent into a domain expert for a chosen industry. Each industry skill compiles the field's top primary sources, sub-skills for representative practitioners, and the reasoning patterns those experts use — so the agent loads one SKILL.md, installs with one curl line, and answers at expert level. The distillation pipeline runs a six-track parallel research swarm followed by multi-stage quality gates; the canonical browsable index lives at master-skill.org, and skills are co-installable into Claude Code, Hermit agents, and OpenClaw.
