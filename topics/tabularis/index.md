**Tabularis** is an open-source, cross-platform database client built around AI and
extensibility. It ships a built-in AI assistant with native [Model Context Protocol
(MCP)](https://modelcontextprotocol.io) support, so models can reason over your data,
run queries and explain results in context. An open plugin ecosystem lets developers
extend the client with new database drivers, UI panels and tools, turning it into a
hackable workbench rather than a closed product. Built with [Tauri](https://tauri.app)
(Rust backend, React frontend) and available in eight languages.
