PhotoDewey is a free photo library manager for Windows. It keeps your original files untouched — all edits, crops, and adjustments are stored separately and applied on the fly, so you can always get back to the original.

Photographers use PhotoDewey to organize large collections with hierarchical keyword tags, browse photos on a map using embedded GPS data, write metadata back to XMP sidecar files, and apply non-destructive edits like tone adjustments, exposure corrections, filters, and rotations. A built-in plugin system lets the community extend the editing tools with additional capabilities.

PhotoDewey stores everything locally — no cloud account required, no files uploaded. The library catalog, thumbnails, and settings all live on your own machine.
